# 프로젝트 소개

KBO 기아 타이거즈의 경기 기록, 리그 순위를 간편하게 볼 수 있는 

### 채팅앱입니다.

야구를 보며 친구들과 혹은 같은 팬과 채팅을 할 때 클릭 한 번으로 볼 수 있으면 좋지 않을까 하는 생각으로 만들었습니다.

소프트웨어 전공으로 처음으로 진행한 프로젝트입니다.

### Flutter와 Firebase를
사용해보며 Flutter와 Firebase의 특징을 공부할 수 있었습니다.

#

#  Project UI 및 주요기능
1. 로그인 회원가입
2. 사용자 간 채팅
   - 텍스트 메시지 전송
   - 이미지 메시지 전송
   - URL 파싱
4. 당일 경기 기록 보기
5. KBO 리그 순위 보기
